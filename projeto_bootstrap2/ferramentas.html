Para a produção dos sites foram usadas as seguintes ferramentas

Visual Studio Code

O Visual Studio Code (VS Code) se tornou uma das ferramentas de desenvolvimento mais populares entre os programadores. Sua versatilidade, personalização e recursos poderosos o tornam uma excelente escolha para uma ampla gama de projetos.

Adobe Color

Através de uma paleta de cores cuidadosamente elaborada, você cria uma identidade visual coesa e memorável, transmitindo a essência da sua marca e conectando-se com seu público.

Bootstrap

Pela a facilidade e diponibilidade de variados modelos do bootstrap a fácil acesso, foi utilizado o bootstrap como ferramenta para a produção dos websites. Não sendo utilizado o ProperJS para a otmização dos websites
